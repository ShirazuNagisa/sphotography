1、边栏默认展开拆分为桌面端 / 移动端两个独立开关：桌面端默认展开、移动端默认收起（以保留全屏地图空间），均可在「主题全局配置 → 边栏信息」中调整。
2、文章展开页顶部 / 底部带有滚动按键的整行新增磨砂渐变过渡带：正文从下方模糊淡入 / 淡出。正文位于最顶端时上边框不做任何模糊，滚动后淡入；底部过渡带常驻。同样的过渡带也应用于友链 / 留言面板。
3、修复微信分享二维码无法加载（破图）：二维码改用内联 SVG 渲染，规避严格 CSP（img-src 无 data:）与图片优化代理对 data-URL 图片的拦截。
4、页面右上角新增「页面链接栏」（横向长条卡片，位于地图缩放 / 指南针图标左侧），包含「友链」「留言」两个入口，以及最多 3 个「外站」入口（后台可自定义名称、链接与悬停文案，点击在新标签页打开）。
5、新增友链页面：从页面链接栏点击「友链」在主页右半部分展开（可与文章展开页共存，仅与留言页互斥），沿用文章展开页的展开动画与磨砂过渡带。友链卡片每行三个，卡片下半部分磨砂过渡中显示站点名称（后台可自填，留空则取站点自带标题）；缩略图默认在后台创建友链后经 mShots 异步抓取并缓存到媒体库，可手动上传覆盖。友链可翻动、可在后台置顶（置顶显示在最上方）。面板最下方为友链申请表单（邮箱 + 链接 + 站点名 + 留言），提交后在后台以待处理列表 + 菜单红点形式提示站长，可一键通过或忽略。
6、评论区新增排序：按时间（可切换正序 / 倒序）与按点赞量排序；置顶评论始终显示在最前，不受排序影响。
7、新增留言页面：从页面链接栏点击「留言」在主页右半部分展开，与友链页互斥（展开其一会先收起另一个）。复用评论系统（点赞、单级回复、Markdown、邮件通知、编辑、IP 属地，无需验证码）。默认随机展示留言，点「展示全部留言」后按时间倒序展示并可切换时间 / 点赞排序。底部磨砂过渡中为分体独立的圆角邮箱输入框、圆角留言输入框与圆形发送按钮；留言框输入到第二行时向上延伸，磨砂过渡同步上移。留言支持 Markdown。
8、评论输入区新增 Markdown 工具栏（粗体 / 斜体 / 删除线 / 行内代码 / 链接 / 引用 / 列表）与可切换的实时预览。
