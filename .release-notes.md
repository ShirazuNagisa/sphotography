1、后台「主题全局配置」恢复为扁平化布局：去掉 v1.3.8 的分类标题，原八大分组的板块改为单列平铺，仅在组与组之间留 1px 分隔线；右侧配置索引保留可展开的二级结构。
2、修复照片墙按键收起时的闪烁：原本 CSS animation-fill-mode: forwards 在 close 动画的 delay 期间回到静态的 opacity:0，造成可见→不可见→可见的闪烁；现改用 Web Animations API（Element.animate）驱动 open/close，按钮显隐完全受 JS 控制，关闭时不再有跳帧。
3、修复照片墙最左一列照片按键被面板左缘遮蔽：原 popup 是 cell 的子节点，受侧边面板的 overflow:hidden + backdrop-filter 堆叠上下文双重裁剪，最左列的按钮会被切掉；现将 popup 挂到 body 上的 #pw-popup-portal 容器中（z-index 9999），用 cell.getBoundingClientRect() 计算坐标并定位，避开面板的 stacking context；最左列的照片自动将 popup 翻到右侧。
