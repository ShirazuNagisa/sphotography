# Sphotography — 宏观图形结构

```
┌──────────────────────────────────────────────────────────┐
│                   全屏地图 (Map Layer)                     │
│  定位: fixed, inset: 0, z-index: 1                       │
│  引擎: MapLibre GL JS + CartoDB Dark Matter / Positron   │
│  内容: GeoJSON 标记点 + 聚合簇                            │
├──────────────────────────────────────────────────────────┤
│                   面板层 (Panel Layer)                     │
│  定位: fixed, z-index: 30 ~ 130                          │
│  样式: 毛玻璃半透明 (blur 24px, bg 0.50)                  │
│  包含: 边栏 | 文章面板 | 照片网格 | 详情 Sheet | 页脚     │
├──────────────────────────────────────────────────────────┤
│                   交互层 (Interaction Layer)               │
│  类型: 地图拖拽/缩放 | 标记点击 | 面板展开/收起 | 滚动    │
│  联动: 边栏 ↔ 文章面板 ↔ 照片网格 ↔ 地图标记点            │
└──────────────────────────────────────────────────────────┘
```

## 五、交互流程图

```
用户操作                   系统响应
─────────                  ────────

访问首页                   → 初始化地图
                            → 加载照片 GeoJSON
                            → 加载边栏文章列表
                            → 显示标记点

拖拽/缩放地图              → 更新照片网格位置 (跟随标记)
                            → idle 时检查聚合/分离
                              → 合并: 显示聚合照片
                              → 分离: 显示独立点照片
                              → 消失: 关闭网格

点击标记点                  → 关闭当前网格
                            → 打开新照片网格 (标记点旁)
                            → 展开边栏 (桌面端)

点击聚合点                  → 获取所有成员照片
                            → 显示在照片网格中

点击网格中照片              → 关闭网格
                            → 展开边栏
                            → 打开文章面板 (桌面)
                            → 打开底部 Sheet (移动端)

点击边栏文章                → 关闭网格
                            → 打开文章面板

收起边栏                    → 关闭文章面板 + 网格 + 详情
                            → 地图占满全屏

展开边栏                    → 恢复边栏

ESC 键                     → 关闭所有面板 (保留地图)

点击地图空白                → 关闭浮动面板 (保留边栏)
```

## 六、响应式断点

```
Desktop (≥1200px)
  │ 边栏: 300px 左侧固定
  │ 文章面板: 340px+ 左侧
  │ 照片网格: 标记点右侧浮动
  │ 作者: 右下 20px/52px

Tablet (768-1199px)
  │ 边栏: 260px 左侧固定
  │ 文章面板: 290px+ 左侧
  │ 照片网格: 标记点右侧浮动
  │ 作者: 右下 20px/52px

Mobile (<768px)
  │ 边栏: 全屏覆盖 (100vw)
  │ 文章面板: 全屏覆盖 (8px 边距)
  │ 照片网格: 底部 Sheet (50vh)
  │ 详情 Sheet: 底部 (70vh)
  │ 作者: 右下 12px/68px
  │ 边栏展开按钮: 缩小至 36×56
```
