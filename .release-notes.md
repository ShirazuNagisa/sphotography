1、公告面板文字重新排版：标题「公告」独占一行、正文另起一行，放宽行距与字间距、明确左对齐、加大四周留白，解决文字上下堆叠拥挤。
2、文章内索引重做并修复：修复点击后消失 / 开局消失 / 点开后无法二次点击（根因是关闭索引时误移除了索引条可见状态）；索引条改为顶端对齐的饱满圆角短条，长度等于未展开任何子索引时的索引高度；展开区随条目动态变化、展开子索引实时向下增长、超出面板高度才内部滚动；展开/收起动画改为索引条向右延展 / 向左收窄；点击条目跳转后索引保持展开，仅在再次点击索引条 / 点别处 / Esc 时收起。
3、「行政区上色」改为默认标记模式；主题激活时后台静默下载边界数据（不阻塞激活），照片定位后增量建索引。
4、新用户装完主题直接访问主页即进入全屏地图，无需手动创建页面/设为首页。
5、修复重建行政区索引目录问题：目录已存在则跳过创建继续；手动放入的边界文件即使缺版本文件也能识别；目录不可写时给出明确 chmod/chown 提示。
6、字体新增「苹方」「宋体」选项（系统字体栈：苹方苹果原生、其余回退；宋体跨平台并以 Noto Serif SC 兜底）。

8、优化载入逻辑：边栏头图按需加载、文章图片惰性加载、地图照片原图改为点开详情时按需获取、精简首屏内联数据。
